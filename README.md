A continuacion una representacion de como se maneja el ciclo
de vida de la aplicacion SAP. Vemos que hay una relacion de estrecha entra DEV (Development) y QAS (Quality Assurance), la retroalimentacion genera cambios (Changes)
que DEV implementa y pasa de nuevo a QAS. Las fases terminadas pasan a PRD (Produccion) donde ellos retroalimentan a QAS y ellos a DEV formandose un circuito de mejora buscando cumplir
con los reqerimientos y la satisfaccion del cliente.
